# WEATHER.app

Projeto Java simples (Maven) para buscar e exibir dados de clima.

Estrutura criada automaticamente. Para compilar e executar:

1. Ajuste `src/main/resources/config.properties` com sua chave de API.
2. Rodar build:

```bash
mvn clean package
```

3. Executar (após build):

```bash
java -cp target/weather-app-0.1.0.jar com.weatherapp.Main
```
